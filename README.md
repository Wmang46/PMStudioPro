# Will Mangaroo - Product Manager Portfolio

Modern portfolio website built with Astro, Tailwind CSS, and MDX.

## Features

- 📝 Hot Topics: Articles and audio about product management
- 🎧 Custom audio player
- 🌓 Dark mode support
- 📱 Fully responsive
- ⚡ Optimized for performance

## Tech Stack

- Astro 6.x
- Tailwind CSS v4
- MDX
- TypeScript
- Vercel

## Local Development

```bash
npm install
npm run dev
```

Visit http://localhost:4321

## Adding Content

Create new MDX file in `src/content/hot-topics/`:

```yaml
---
title: "Article Title"
type: "article"
date: 2026-03-10
tags: ["Tag"]
summary: "Summary..."
---
```

## Deployment

Automatic via Vercel on push to main.
